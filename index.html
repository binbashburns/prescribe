PRESCRIBE Architecture Explorer

Post-exercise Readiness & Exercise-Sourced Competency Remediation, Informed By Evidence

PRESCRIBE is a graduate capstone project exploring how to apply commercial data-platform patterns to post-exercise cyber readiness assessment. It ingests post-exercise artifacts from cyber EXEVALs (After Action Reports, evaluator scorecards, hot wash notes, range logs, scan output), AI-extracts behavior observations tagged to DCWF KSAs, computes per-operator competency gaps, and generates individualized remediation plans.

This site is the design tool, not the build itself. The actual capstone is implemented separately on a commercial data platform. The four tabs at the top each cover one architectural layer:

  • Data processing. batch pipelines that read from S3 and external catalogs, parse heterogeneous formats, and write into the ontology.
  • Ontology. The 19 object types and their relationships - what data PRESCRIBE works with at rest.
  • AI Automate. The handful of AI Function that turn unstructured text into structured ontology rows. Each card opens a paste-ready prompt.
  • Application. Sketches of the App views that consume the ontology - operator profile, mentor validation queue, CPT EXEVAL summary, and so on.
All data shown is fictional. Sample callsigns, unit designators (e.g., 666th CPB, EXERCISE-DEMO), IPs (10.10.x.x, 198.51.100.x), and adversary domains (c2.example.adversary) are placeholder values used for design illustration. Not classified or operational.
Public frameworks this design draws on
  • DoD Cyber Workforce Framework (DCWF) - the competency spine
  • NIST NICE Framework - cybersecurity workforce KSAs
  • NICCS Education & Training Catalog (CISA) - external course corpus
  • DoD 8140 Qualification Matrices - work-role-tagged courses + certifications
  • AFDP 3-12, Cyberspace Operations - cyberspace doctrine
  • MITRE ATT&CK - technique tagging reference
